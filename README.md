# Reona's dotfiles

The dotfiles updated from time to time.
Editor: NeoVim; Shell: zsh; Terminal: Alacritty, tmux; OS: MacOS Monterey

## Usage

After cloning this repository, execute the following command.

```shell
$ make all
```
